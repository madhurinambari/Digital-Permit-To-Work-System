# 🚧 Digital Permit-to-Work (PTW) System

A full-stack web application developed to digitize and automate the **Permit-to-Work (PTW)** process in industrial environments. The system replaces traditional paper-based permits with a secure digital workflow, improving workplace safety, operational efficiency, and transparency.

---

## 📌 Project Overview

The Digital Permit-to-Work (PTW) System enables organizations to manage hazardous work permits through an online platform. It provides role-based access, digital permit creation, approval workflows, real-time tracking, and centralized record management.

---

## ✨ Features

- 🔐 User Authentication & Authorization
- 👨‍🏭 Role-Based Access Control
- 📝 Digital Permit Creation
- ✅ Multi-Level Approval Workflow
- ⚠️ Risk Assessment Management
- 📊 Dashboard & Analytics
- 🔔 Permit Status Tracking
- 📁 Centralized Permit Records
- ☁️ Cloud Deployment using Render

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/madhurinambari/Digital-Permit-To-Work-System--PTW-.git
```

### Backend Setup

```bash
cd ptw-backend
npm install
npm start
```

### Frontend Setup

```bash
cd ptw-frontend
npm install
npm run dev
```
